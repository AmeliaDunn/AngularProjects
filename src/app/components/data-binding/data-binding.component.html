Data Binding


Interpolation

  • String interpolation inserts interpolated strings into the text within HTML Elements
  • String interpolation uses double curly brackets and NO QUOTES
  • {{ interpolatedString }}
  • {{ interpolatedNumber }}
  • {{ interpolatedBoolean }}

Property Binding

  • Property binding sets an element's property to the value of a Template Expression
  • The binding target is enclosed in square brackets []
  • The binding source is enclosed in single quotes ''
